## Installation

Web Client
```
npm install
```
AI Server
```
cd server
npm install
```

## Usage

To start the project:

Web Client
```
npm start
```
AI Server
```
cd server
npm start
```

You start by describing the problem, the input format and the expected output format. Then click the "train" button. You can improve the solution by clicking the "improve" button.
